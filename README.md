# Saarthi4u - Backend

Official repository for the backend of Saarthi4u.

## Prerequisites

- Node.js (v22.11.0 stable)  - [Download Link](https://nodejs.org/en/blog/release/v22.11.0)
- Xaammp or MariaDB  - [Download Link](https://www.apachefriends.org/index.html) or [Download Link](https://mariadb.org/download/)
- Postman (for API testing & API documentation) - [Download Link](https://www.postman.com/downloads/)

## Steps to Run the Project

In the project directory, you can run:

- ### `npm i` or `npm install` - only first time

- ### `npm run start` - to run backend

- ### `npm run stop` - to stop backend - everytime before running start use this command
